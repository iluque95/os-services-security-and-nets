Pràctica 1 - Configuració de les interfícies de xarxa Wi-Fi

1- Configurar les interfícies Wi-Fi d'un servidor Debian. La interfície Wi-Fi s'ha de connectar a una xarxa Wi-Fi amb seguretat wpa2.

2- Programar un script "informa_wifi.sh" amb shell que proporcioni la següent informació en un fitxer de sortida "info_wifi.txt".
    - El format de sortida cal que sigui "nom dada:    valor"
    - Dades de l'equip
        Data
        Usuaris humans
        Nom de l'equip
        Adreça IP del router
        Adreça IP externa
        Adreces IP dels DNS
    - Dades de les interfícies Wi-Fi
        Nom de la interfície
        Adreça MAC
        Protocols suportats
        Si està connectada a una xarxa   
            Adreça IP
            Nom DNS
            Màscara de xarxa
            Adreça de broadcast
            Adreça de xarxa
            Nom de la xarxa
            MTU
    - Xarxes Wi-Fi operatives (per cada una)
        Adreça MAC AP
        Nº de canal
        Qualitat
        Nivell de senyal
        ESSID
        Xifrat
   
3- Lliurar els resultats mitjançant Atenea (2 fitxers).
    - Redactar l'informe de la pràctica p3_cognom1_nom.txt.
    - Realitzar les proves necessàries per justificar els resultats i encapsular els fitxers necessaris en el fitxer p3_cognom1_nom.zip.
